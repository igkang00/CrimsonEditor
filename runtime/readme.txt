Crimson Editor 3.80
===================

A freeware text editor for Windows.
Originally developed 1999-2005 by Ingyu Kang, modernized in 2026
on Visual Studio 2026 (v145 toolset) for current Windows.

Project page : https://github.com/igkang00/CrimsonEditor
License      : See LICENSE.txt (MIT)
